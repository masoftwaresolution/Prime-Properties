# 🏡 Prime Properties – Real Estate Website

A modern and fully responsive Real Estate website developed using **React.js**, **Vite**, and **Tailwind CSS**. The project features a clean and professional dark-themed user interface designed to provide an excellent user experience.

## ✨ Features

- 🏠 Modern Home Page
- 🏢 Property Categories
- ⭐ Featured Properties
- 📋 Property Listings
- 🔍 Search Bar
- 📖 About Us
- 🛠 Our Services
- 👨‍💼 Our Team
- 📞 Contact Us
- ❓ FAQ Section
- 🔐 Login & Signup Pages
- 📱 Fully Responsive Design
- 🎨 Smooth Hover Effects & Animations
- 💬 Property Details Modal

## 🛠 Technologies Used

- React.js
- Vite
- Tailwind CSS
- React Router DOM
- Lucide React Icons
- React Icons
